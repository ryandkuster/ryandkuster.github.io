# Introduction to Command Line

This module is a practical introduction to working in the Unix shell. It follows the structure of the [Software Carpentry Unix Shell lesson](https://swcarpentry.github.io/shell-novice/) and uses files from the `practice_data/` directory for hands-on exercises.

## Sections

1. [Introducing the Shell](01-intro/)
2. [Navigating Files and Directories](02-filedir/)
3. [Working With Files and Directories](03-create/)
4. [Pipes and Filters](04-pipefilter/)
5. [Loops](05-loop/)
6. [Shell Scripts](06-script/)
7. [Finding Things](07-find/)


This lesson is an adaptation of the Software Carpentry 'The Unix Shell' lesson ([CC-BY 4.0](https://creativecommons.org/licenses/by/4.0/)), with modifications for EPP 575 : Introduction to RNA-Seq.
